2. üìù The Golden Prompt Formula ‚Äã

Context + Action + Output Format

Bad: "Make a landing page." Good: "Context: We are building a portfolio for a designer. Action: Create index.html with a hero section, grid gallery, and contact form. Use TailwindCSS. Output: Only the code for index.html."

3. üß† Context Management ‚Äã

Rule of Thumb:

  • Start Fresh: Reset context every major feature.
  • Reference: Point to existing files (@src/utils.ts) instead of pasting entire repo.
  • Docs: Keep a tech_stack.md or .cursorrules to ground the AI.
