## üìñ About This Repo

**Weekend-to-Release** is a practical, ELI-15 guide to turning a "weekend idea" into a real, professional release.

If you've ever built a cool script or app but stalled on packaging, documentation, versioning, or support, this course is for you. We focus on the **boring-but-critical** parts that make shipping possible:

- üì¶ **Packaging & Installation**

- üß™ **Testing without Tears**
- üè∑Ô∏è **Versioning & Changelogs**
- üõ°Ô∏è **Support & Maintenance**

> **üí° Quick Start:** Check out the **[2-Page Visual Cheatsheet](../CHEATSHEET.md)** for the high-level roadmap.

This repo itself is the course. You don't need to sign up anywhere. Just clone it, read the modules, and follow the checklists.
