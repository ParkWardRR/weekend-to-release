## üìñ About This Repo

**Weekend-to-Release** is a dual-track curriculum:
1.  **Magical Construction (80%)**: How to use AI Agents (`claude.code`, Google Antigravity) to build complex software without getting stuck.
2.  **Professional Release (20%)**: The rigorous "boring stuff" (packaging, docs, versioning) that turns a hacky script into a real product.

### The Problem
Most AI-generated code stays in the chat window. It's often buggy, unmaintainable, or "works on my machine" only.

### The Solution
We treat the AI as a **Junior Developer** and you as the **Engineering Manager**. You provide the Spec, the Review, and the Release Process. The AI writes the code.
